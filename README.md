## 项目布局
```
├── components                                  // 组件
├── images                                      // 图片
├── pages                                       // 页面
│   ├── detail                                  // 详情页
│   ├── epicure                                 // 美食家主页
│   ├── joke                                    // 开心时刻详情页
│   ├── list                                    // 我的收藏页
│   ├── location                                // 到店咨询
│   ├── newsDetail                              // 文字详情页
│   ├── tabBar                                  // tabBar选项
│   │   ├── fl                                  // 分类
│   │   ├── index                               // 首页
│   │   ├── look                                // 发现
│   │   └── wd                                  // 我的
│   └── userinfo                                // 编辑个人资料
├── styles                                      // weui样式
├── utils                                       // 入口html文件
```
